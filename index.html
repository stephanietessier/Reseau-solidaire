🤝 Réseau de solidarité locale IA

Bienvenue sur le projet Réseau de solidarité locale IA.

Cette application utilise l'intelligence artificielle pour mettre en relation les besoins des citoyens avec les ressources disponibles dans leur communauté.

Objectif

  • Trouver de l'aide rapidement.
  • Mettre en relation les personnes.
  • Favoriser la solidarité locale.

Le code source est disponible sur GitHub.
